<h2><p align="center">US Healthcare Data Analysis & Dashboard</p></h2>

<h3>📊 Project Overview</h3>

This project focuses on analyzing US healthcare data to derive actionable insights for patient pricing optimization and hospital resource management. It involves end-to-end data processing from raw data collection to dashboard visualization.


<h3>⚙️ Tech Stack</h3>

SQL (MySQL / SQL Server)
Excel (Data Cleaning & Feature Engineering)
Power BI / Excel Dashboards
Data Analysis & Visualization

<h3>🔄 Workflow</h3>

1. Business Understanding
Identified key healthcare problems:
Patient cost optimization
Resource utilization

2. Data Collection
Sources:
SQL databases
CSV / Excel files

3. Data Cleaning
Corrected inconsistent values (e.g., Age “8l” → 81)
Standardized gender values (M → Male)
Fixed admission type inconsistencies
Converted data types (text → numeric)
Corrected billing errors (O → 0)
Rectified date inconsistencies (49 records)

4. Feature Engineering
Created Age Buckets:
Young (<35)
Middle (35–60)
Senior (>60)
Built Demographic Segments (Age + Gender)
Extracted Year from Admission Date
Calculated Length of Stay (No. of Days)

5. Data Analysis
Performed exploratory data analysis (EDA)
Built pivot tables for trend analysis
Identified patterns in: Patient demographics, Admission types, Billing amounts

6. Dashboard Development
Created interactive dashboards showing:
Patient distribution
Revenue insights
Resource usage

<h3>📈 Key Insights</h3>

Senior patients contribute higher hospitalization duration
Emergency admissions show higher billing variability
Certain demographics drive higher revenue

<h3>🎯 Business Impact</h3>

Improved decision-making for pricing strategies
Enhanced hospital resource allocation
Enabled data-driven healthcare insights
